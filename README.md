## Continuous JS- and TS- Script Executor (CSE)

- Executes JS-script [index.js](index.js) continuously on change.
- Compiles TS-script [index.ts](ts/index.ts) continuously on change to `ts/index.js` and executes it.

### Installation

```
npm install
```

### Start

```
npm start
```
